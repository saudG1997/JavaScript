# Arrays
## Arrays are variables which can hold more than one value.
        const fruits = ["banana","apple","grapes"]
        const a1 = [7, "Harry", false]      // can be different types

## Accessing values
        let numbers = [1,2,7,9]
        numbers[0]      // o/p:1
        numbers[1]      // o/p:2

## Finding the length
        let numbers = [1,7,9,21]
        numbers[0]          // o/p:1
        numbers.length      // o/p:4
